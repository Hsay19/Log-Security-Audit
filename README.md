# Log-Security-Audit
这是一个专业的SSH安全审计脚本，专门用于对Linux系统进行全面的安全检查和风险评估。脚本通过多维度安全检查，帮助系统管理员快速发现潜在的安全隐患和配置问题。

✨ 功能亮点

🔐 账户安全审计
异常账户检测（UID异常、属组异常）
特权用户审查（UID为0的用户）
可登录账户分析
Sudo权限审计
账户登录状态检查

🔍 登录安全分析
成功登录记录分析（最近30条）
失败登录尝试监控
实时登录用户查看
认证日志关键事件提取

🌐 网络安全检查
当前网络连接状态
开放端口与服务识别
监听端口分析
网络连接详细信息

⚡ 进程资源监控
高CPU使用进程TOP15
高内存占用进程TOP15
长时间运行进程识别
SSH相关进程检查

⏰ 持久化机制审计
系统计划任务审查
用户自定义cron任务
启动项和服务检查
系统初始化脚本审计

📁 文件权限验证
关键系统文件权限检查
SSH配置文件权限验证
密钥文件安全状态

📊 智能安全评分
自动化安全风险评估
多维度评分体系
具体安全警告提示
整体安全状况可视化

🚀 使用优势
🎯 专业全面
覆盖SSH安全的各个方面
基于行业最佳实践
多维度深度检查

⚡ 高效便捷
一键自动化审计
实时彩色输出
进度状态可视化

📈 结果清晰
结构化报告输出
风险等级分类
具体修复建议

🛡️ 安全可靠
只读检查操作
无系统修改
完整的执行日志
